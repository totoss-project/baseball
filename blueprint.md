
# 로또 번호 추첨기

## 개요

이 프로젝트는 사용자가 버튼을 클릭하여 1부터 45까지의 숫자 중 6개의 고유한 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다.

## 기능 및 디자인

*   **번호 생성:** 버튼 클릭 시 6개의 고유한 난수를 생성합니다.
*   **번호 표시:** 생성된 번호를 시각적으로 매력적인 방식으로 화면에 표시합니다.
*   **UI:**
    *   중앙 정렬된 레이아웃.
    *   생성된 각 번호를 위한 스타일링된 원.
    *   "번호 생성" 버튼.
    *   현대적인 글꼴, 색상 및 그림자 효과.

## 현재 계획

1.  **`index.html` 수정:** 제목을 변경하고 로또 번호 표시 영역과 생성 버튼을 추가합니다.
2.  **`style.css` 수정:** 애플리케이션의 전체적인 모양과 느낌을 스타일링합니다.
3.  **`main.js` 수정:** 로또 번호 생성 및 표시 로직을 구현합니다.
